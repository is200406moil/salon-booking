# Руководство пользователя — Администратор

## Вход в админ‑панель
1. Откройте `/admin/`.
2. Введите логин и пароль администратора.

## Управление услугами
1. Откройте раздел «Услуги» в админ‑панели или страницу `/services/admin/`.
2. Добавьте новую услугу (название, длительность, цена, описание).
3. При необходимости редактируйте или отключайте услуги.

## Управление мастерами
1. Откройте раздел «Мастера» (`/masters/admin/`).
2. Создайте профиль мастера и привяжите пользователя.
3. Укажите список услуг, которые выполняет мастер.

## Настройка рабочих слотов
1. Перейдите в раздел «Рабочие слоты» (`/masters/slots/`).
2. Добавьте интервалы времени для мастеров.
3. Слоты автоматически становятся недоступными после записи клиента.

## Просмотр записей
1. Перейдите в «Все записи» (`/appointments/admin/list/`).
2. Контролируйте загрузку салона и статус визитов.

## Выход
Нажмите «Выйти» в верхнем меню — вы будете перенаправлены на страницу входа.
