# Руководство пользователя — Мастер

## Вход
1. Перейдите на страницу входа: `/accounts/login/`.
2. Введите данные аккаунта мастера.

## Просмотр расписания
1. Откройте страницу «Расписание мастера».
2. В блоке «Записи» отображаются клиенты и услуги.
3. В блоке «Доступные слоты» виден статус временных интервалов.

## Обновление профиля
Мастер может обновить контактные данные на странице профиля.
