# Документация разработчика

## Веб-приложение «Электронная запись в салон красоты»

**Версия документации:** 1.0 (реализация)

### Назначение системы
Система предназначена для автоматизации записи клиентов на услуги салона красоты,
управления расписанием мастеров и контроля загруженности салона.

### Ключевые сценарии
- Регистрация и авторизация клиентов.
- Подбор услуги, мастера и свободного слота.
- Запись, перенос и отмена визита.
- Просмотр расписания мастерами.
- Управление услугами, мастерами и слотами администратором.

### Архитектура
Проект реализован на Django 5.x в MVT-архитектуре и разделён на приложения:
`accounts`, `services`, `masters`, `appointments`. База данных — PostgreSQL.
